# Tugas C - Bias, Fairness, dan CoT

App ini mengubah skrip Python biasa menjadi aplikasi Hugging Face Spaces berbasis Gradio.

## Fitur

1. Analisis bias dan stereotip untuk daftar kalimat
2. Kalkulator Disparate Impact
3. Perbandingan jawaban tanpa CoT vs dengan CoT

## Secret yang dibutuhkan

- `GROQ_API_KEY`
- Opsional: `GROQ_MODEL`

## Jalankan lokal

```powershell
pip install -r requirements.txt
python app.py
```
